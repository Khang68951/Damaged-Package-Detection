# Package Damage Detection

Real-time package damage detection using Faster R-CNN with Flask API and web interface.

## Requirements

- Python 3.7+
- Webcam (for camera interface)
- Model file: `best_model.pth`

## Installation

1. **Install Python dependencies:**
pip install flask flask-cors torch torchvision pillow

2. **Place model file in root directory:**
Copy best_model.pth to the same folder as app.py

3. **Run the application:**
python app.py
